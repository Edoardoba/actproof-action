### â“ FAQ

#### Q: Is my source code stored?
**A:** In local mode (default), no code leaves the GitHub runner. In API mode, code is temporarily processed and then deleted.

#### Q: Is the action free?
**A:** Yes! The basic action is free to use. Premium features may be available in the future.

#### Q: Does using this guarantee EU AI Act compliance?
**A:** No. ActProof.ai provides guidance and recommendations. Final compliance requires professional legal review.

#### Q: Can I use this for private repositories?
**A:** Yes! Local mode works with both public and private repositories.

#### Q: What languages are supported?
**A:** Currently Python and JavaScript. More languages coming soon.
